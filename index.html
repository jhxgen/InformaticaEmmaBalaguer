Oportunidades Globales

Salarios Competitivos

Plan de Estudio

Primer Año(4to): Fundamentos basicos

  • Desarrollo web Básico (HTML, CSS, ETC)
  • Analisis y diseño de Sistemas
  • Introducción a Bases de Datos

Segundo Año(5to): Fundamentos avanzados

  • Estructuras de Datos
  • Programación Orientada a Objetos
  • Fundamentos avanzados de Bases de Datos

Tercer Año(6to): Especialización

  • Implementacion y mantenimiento de sistemas informaticos
  • Desarrollo Web Avanzado
  • Desarrollo y administracion de sistemas informaticos
